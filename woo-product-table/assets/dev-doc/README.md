## 🔍 কোড উদাহরণ

### একটি সিম্পল কলাম তৈরি

```php
// কলাম রেজিস্ট্রার করুন
add_filter( 'wpto_default_column_arr', function( $columns ) {
    $columns['my_column'] = 'আমার কলাম';
    return $columns;
});

// টেমপ্লেট লোকেশন
add_filter( 'wpto_template_loc_item_my_column', function( $file ) {
    return __DIR__ . '/templates/my-column.php';
});
```

### হুক ব্যবহার করুন

```php
// টেবিলের আগে কিছু দেখান
add_action( 'wpt_before_table', function( $table_id ) {
    echo '<div class="notice">বিশেষ অফার!</div>';
});

// দাম পরিবর্তন করুন
add_filter( 'wpt_product_price', function( $price, $product ) {
    return $price . ' টাকা';
}, 10, 2 );
```
