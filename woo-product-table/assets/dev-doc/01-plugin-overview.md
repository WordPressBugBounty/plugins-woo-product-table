# প্লাগিন ওভারভিউ - Woo Product Table

## প্লাগিন সম্পর্কে

**Woo Product Table** হলো একটি WooCommerce প্লাগিন যা আপনার পণ্যগুলিকে একটি সুন্দর টেবিল ফরম্যাটে প্রদর্শন করতে সাহায্য করে। এটি বিশেষভাবে পাইকারি ব্যবসার জন্য উপযোগী।

## মূল তথ্য

- **প্লাগিনের নাম:** Product Table for WooCommerce
- **বর্তমান ভার্সন:** 5.0.6
- **ডেভেলপমেন্ট ভার্সন:** 5.0.6.2
- **টেক্সট ডোমেইন:** woo-product-table
- **প্রয়োজনীয় প্লাগিন:** WooCommerce

## প্লাগিনের প্রধান বৈশিষ্ট্য

1. **সহজ শর্টকোড ব্যবহার:** 
   - মাত্র `[Product_Table id='123']` লিখে যেকোনো পেজে টেবিল দেখানো যায়

2. **কাস্টম পোস্ট টাইপ:**
   - `wpt_product_table` নামের একটি কাস্টম পোস্ট টাইপ তৈরি করা হয়েছে
   - প্রতিটি টেবিল একটি পোস্ট হিসেবে সংরক্ষিত থাকে

3. **বিভিন্ন ধরনের কলাম:**
   - থাম্বনেইল, টাইটেল, দাম, স্টক, রেটিং ইত্যাদি
   - কাস্টম ফিল্ড এবং ট্যাক্সোনমি সাপোর্ট

4. **ফিল্টারিং এবং সার্চিং:**
   - টেবিলে সার্চ বক্স
   - বিভিন্ন ফিল্টার অপশন

5. **রেসপন্সিভ ডিজাইন:**
   - মোবাইল এবং ট্যাবলেটে সুন্দরভাবে কাজ করে

6. **AJAX সাপোর্ট:**
   - পেজ রিলোড ছাড়াই ডেটা লোড হয়

## প্রযুক্তিগত স্ট্যাক

### ফ্রন্টএন্ড
- **JavaScript Libraries:**
  - DataTables - টেবিল ফাংশনালিটির জন্য
  - Select2 - ড্রপডাউন এবং ফিল্টারের জন্য
  - jQuery - মূল JavaScript লাইব্রেরি

### ব্যাকএন্ড
- **PHP Version:** 7.4+
- **WordPress Version:** 6.2+
- **WooCommerce Version:** 6.2.2+
- **Namespace:** `WOO_PRODUCT_TABLE`

### বিল্ড সিস্টেম
- **Node.js & NPM:** জাভাস্ক্রিপ্ট ডিপেন্ডেন্সি ম্যানেজমেন্টের জন্য
- **Package.json:** বিল্ড স্ক্রিপ্ট এবং ডিপেন্ডেন্সি

## কোড স্ট্রাকচার

প্লাগিনটি OOP (Object Oriented Programming) প্যাটার্নে তৈরি করা হয়েছে এবং PHP Namespace ব্যবহার করা হয়েছে।

### প্রধান ক্লাস

1. **WPT_Product_Table** - মূল প্লাগিন ক্লাস
2. **WOO_PRODUCT_TABLE\Inc\Shortcode** - শর্টকোড হ্যান্ডলিং
3. **WOO_PRODUCT_TABLE\Inc\Table\Row** - টেবিল রো জেনারেশন
4. **WOO_PRODUCT_TABLE\Admin\Admin_Loader** - অ্যাডমিন এরিয়া

## ডেটা স্টোরেজ

### কাস্টম পোস্ট টাইপ
- **পোস্ট টাইপ:** `wpt_product_table`
- **মেটা ডেটা:** টেবিলের সব কনফিগারেশন পোস্ট মেটায় সংরক্ষিত থাকে

### গ্লোবাল অপশন
- **অপশন কী:** `wpt_configure_options`
- এখানে প্লাগিনের গ্লোবাল সেটিংস থাকে

## ফাইল স্ট্রাকচার সারাংশ

```
woo-product-table/
├── woo-product-table.php    # মূল প্লাগিন ফাইল
├── admin/                   # অ্যাডমিন এরিয়ার কোড
├── inc/                     # মূল শর্টকোড এবং ফিচার
├── includes/                # পুরাতন শর্টকোড (ব্যাকওয়ার্ড কম্প্যাটিবিলিটি)
├── assets/                  # CSS, JS, ইমেজ ফাইল
├── templates/               # HTML টেমপ্লেট
├── core/                    # কোর ফাংশনালিটি
├── framework/               # ফ্রেমওয়ার্ক ফাইল
├── premium/                 # প্রিমিয়াম ভার্সন কোড
└── languages/               # অনুবাদ ফাইল
```

## ডেভেলপমেন্ট ওয়ার্কফ্লো

1. **ইন্সটলেশন:** `npm install`
2. **ডেভেলপমেন্ট:** কোড পরিবর্তন করুন
3. **জিপ তৈরি:** `npm run plugin-zip`
4. **Freemius ডিপ্লয়:** `npm run fs`

## হুক সিস্টেম

প্লাগিনে প্রচুর পরিমাণে হুক (Filter এবং Action) রয়েছে যা দিয়ে ফাংশনালিটি এক্সটেন্ড করা যায়।

### গুরুত্বপূর্ণ হুকস

- `wpto_default_column_arr` - নতুন কলাম যোগ করার জন্য
- `wpt_avialable_variables` - আইটেম ফাইলে ভেরিয়েবল পাস করার জন্য
- `wpto_column_setting_form_{keyword}` - কলাম সেটিংস ফর্মের জন্য
- `wpto_template_loc_item_{keyword}` - কাস্টম টেমপ্লেট লোকেশন
